Important libraries and packages
********************************

- `Numpy <https://numpy.org/>`_ ; `pip <https://pypi.org/project/numpy/>`_
    - pip install numpy
    - basic library for numeric data

- `SymPy <https://www.sympy.org/en/index.html>`_ ; `pip <https://pypi.org/project/sympy/>`_
    - pip install sympy
    - basic library for symbolic data, contains complete modules for, e.g., `continuum mechanics <https://docs.sympy.org/latest/modules/physics/continuum_mechanics/index.html?highlight=mechanics>`_

- `SciPy <https://scipy.org/>`_ ; `pip <https://pypi.org/project/scipy/>`_
    - pip install scipy
    - library for scientific computing

- `Matplotlib <https://matplotlib.org/>`_ ; `pip <https://pypi.org/project/matplotlib/>`_
    - pip install matplotlib
    - library for plots

- `Pandas <https://pandas.pydata.org/>`_ ; `pip <https://pypi.org/project/pandas/>`_
    - pip install pandas
    - library for data treatment

- `Jupyter Notebook <https://jupyter.org/>`_ ; `pip <https://pypi.org/project/jupyter/>`_
    - pip install jupyter
    - environment to build a CAS with SymPy and Ipython

- `Sphinx <https://www.sphinx-doc.org/en/master/>`_ ; `pip <https://pypi.org/project/Sphinx/>`_
    - pip install Sphinx
    - useful tool for automated code and (small) project documentation
