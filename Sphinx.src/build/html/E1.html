Get ready for the course

The following section introduces the tools and libraries used in the course.

Tools

  • Python 3.9.11

    • for this course we use still Python 3.9 because it supports all used libraries without problems

  • PyCharm Community Edition

    • YouTube Series from JetBrains : Getting started with PyCharm

    • Turn on PEP8! ; PEP8 Style Guide

  • GitHub Desktop

    • YouTube Tutorial

Important libraries and packages

  • Numpy ; pip

    • pip install numpy

    • basic library for numeric data

  • SymPy ; pip

    • pip install sympy

    • basic library for symbolic data, contains complete modules for, e.g., continuum mechanics

  • SciPy ; pip

    • pip install scipy

    • library for scientific computing

  • Matplotlib ; pip

    • pip install matplotlib

    • library for plots

  • Pandas ; pip

    • pip install pandas

    • library for data treatment

  • Jupyter Notebook ; pip

    • pip install jupyter

    • environment to build a CAS with SymPy and Ipython

  • Sphinx ; pip

    • pip install Sphinx

    • useful tool for automated code and (small) project documentation
