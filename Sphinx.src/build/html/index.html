Welcome to MEC640’s documentation!

The course: Temas de Diseño en Ingeniería Mecánica D - Aplicación de métodos matemáticos para ingeniería implementado por Python should facilitate the students to solve tasks in scientific computing beyond extensive software frameworks like ANSYS .

Therefore, they are going to solve several exercise of well known problems by implementing them in Python, any created source code will be documented on this website for later review, reuse etc. The source files are available at GitHub too.

Schedule

The class is scheduled from 10:00 am to 01:00 pm GMT-5 (peruvian time).
